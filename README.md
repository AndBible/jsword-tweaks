# JSword-tweaks

jsword-tweaks is used to build jsword.jar for And Bible.

# Preparations

First get JSword from https://github.com/AndBible/jsword
and And Bible from https://github.com/AndBible/and-bible

# How to build jsword.jar?

Run 

```
TARGET_DIR=/path/to/and-bible/app/libs JSWORD_PATH=/path/to/jsword ./gradlew install
```
